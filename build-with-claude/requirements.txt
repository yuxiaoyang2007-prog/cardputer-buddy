# Apache-2.0-compatible) so a fresh clone has serial support without

# `m5-onboard` skill's preflight auto-installs it on first run if
# it's missing from the user's environment, so attendees with
# unconfigured machines still get a one-command experience.
#
# To install everything explicitly (CI, reproducible setup):
#   python3 -m pip install --user -r requirements.txt
